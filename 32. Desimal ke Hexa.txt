Buatlah program untuk mengonversi bilangan desimal ke bilangan heksadesimal (basis 16).

Spesifikasi Input
Satu baris berisi satu buah bilangan bulat positif X (1 ≤ X ≤ 231 – 1).

Spesifikasi Output
Satu baris berisi representasi X dalam bilangan heksadesimal tanpa awalan digit 0.

For example:

Input
53
26

Result
35
1A
